# 🚀 Royal Maricay Hotel - Quick Deployment Guide

## The Fastest Way to Deploy (5 minutes)

### 1️⃣ DEPLOY WITH NETLIFY (Easiest)

**Step 1:** Go to [netlify.com](https://netlify.com) and sign up (free)

**Step 2:** Click "Add new site" → "Deploy manually"

**Step 3:** Drag and drop the entire project folder

**Step 4:** Done! Your site is live

**Get Your Domain:**
- Free: yourdomain.netlify.app
- Custom: $12/year (point your domain to Netlify)

---

### 2️⃣ DEPLOY WITH VERCEL (Best Performance)

**Step 1:** Go to [vercel.com](https://vercel.com) and sign up (free)

**Step 2:** Click "New Project"

**Step 3:** Upload your project folder

**Step 4:** Click "Deploy"

**Your site is instantly live!**

---

### 3️⃣ DEPLOY ON GITHUB PAGES (Free Forever)

**Step 1:** Go to [github.com](https://github.com) and sign up (free)

**Step 2:** Create new repository called `royal-maricay-hotel`

**Step 3:** Upload all files to repository

**Step 4:** Go to Settings → Pages → Select "main" branch

**Step 5:** Your site is live at: `yourusername.github.io/royal-maricay-hotel`

---

### 4️⃣ CUSTOM DOMAIN (Optional)

For any platform above:

1. Buy domain from GoDaddy, Namecheap, etc. (~₵30-50/year)
2. Update domain DNS settings (your hosting will give instructions)
3. Wait 24 hours for DNS propagation
4. Your site is now at www.royalmaricayhotel.com

## 🔧 Troubleshooting

**"My site doesn't load"**
- Check internet connection
- Wait 24 hours for DNS to update
- Try different browser

**"WhatsApp button doesn't work"**
- It opens WhatsApp app on mobile ✅
- On desktop, it opens web.whatsapp.com
- Make sure phone number is correct

**"Contact form doesn't send email"**
- Currently only shows success message
- To send actual emails, need backend (future upgrade)
- WhatsApp is recommended alternative

**"Images aren't showing"**
- Check image file paths
- Ensure image files are in correct folder
- Use proper file names (no spaces)
